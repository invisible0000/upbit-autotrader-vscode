📊 Upbit AutoTrader 데이터베이스 디렉토리
==============================================

이 디렉토리는 업비트 자동매매 프로그램의 데이터베이스 파일들을 저장합니다.

📁 기본 데이터베이스 파일들:
---------------------------
1. settings.sqlite3     - 프로그램 설정 및 사용자 환경설정
2. strategies.sqlite3   - 매매 전략 및 실행 데이터
3. market_data.sqlite3  - 시장 데이터 및 차트 정보

💡 다중 프로필 관리:
------------------
파일명에 날짜나 사용자명을 추가하여 여러 버전을 관리할 수 있습니다.

예시:
- settings_20250728_user1.sqlite3
- strategies_backup_20250728.sqlite3
- market_data_production.sqlite3

⚙️ 설정 방법:
-----------
1. 메인 화면에서 '설정' 클릭
2. '데이터베이스' 탭 선택
3. 각 DB 파일의 '찾아보기' 버튼으로 파일 선택
4. '데이터베이스 교체' 버튼으로 적용

⚠️ 주의사항:
----------
- 데이터베이스 변경 시 프로그램이 재시작됩니다
- 중요한 데이터는 백업을 권장합니다
- 손상된 DB 파일은 프로그램 오류를 발생시킬 수 있습니다

📦 설치/배포 정보:
---------------
프로그램 최초 실행 시 기본 DB 파일들이 자동으로 생성됩니다.
설치 프로그램은 이 디렉토리와 기본 파일들을 포함하여 배포됩니다.
