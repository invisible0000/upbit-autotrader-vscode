## 🎨 UI 설계 참고

### 2x2 그리드 레이아웃
```
┌─────────────────┬─────────────────┐
│ 전략 선택 영역    │ 조합 설정 영역    │
│ (진입/관리전략)   │ (충돌해결/우선순위)│
├─────────────────┼─────────────────┤
│ 미리보기 영역     │ 제어 영역        │
│ (조합 시각화)     │ (저장/백테스트)  │
└─────────────────┴─────────────────┘
```

### 상태 표시 시스템
- 🟢 VALID: 유효한 조합
- 🟡 WARNING: 주의 필요
- 🔴 INVALID: 무효한 조합
